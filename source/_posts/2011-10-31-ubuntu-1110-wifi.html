Recently I upgraded from Linux Mint (Julia) to Ubuntu 11.10. I did so by creating a live USB of the Ubuntu 11.10 iso. Took backup of my stuff from the Mint partition, formatted it and installed Ubuntu over it.
It is a welcome change. Though the Unity interface requires some getting used to.
One of the first things I tried after installing is connecting to the internet. Initially the WiFi in Ubuntu 11.10 was not working. Here is how I got the WiFi working.

Generally when one installs Ubuntu, the WiFi drivers can be found in the 'Additional Drivers' and activated from there. But in my case, it was not able to find any driver. When I clicked on the network icon on the top panel, it showed "Wireless is disabled".
I tried googling this, got some tips but none of them helped.

So I downloaded bcmwl-kernel-source package from here on a friend's laptop. Tried to install it on my laptop.
dpkg -i bcmwl-kernel-source_5.100.82.38+bdcom-0ubuntu4_amd64.deb
This showed that a few dependents were missing. So downloaded dkms and patch ie the missing packages.
dpkg -i patch_2.6.1-2_amd64.deb
dpkg -i dkms_2.2.0.2-1ubuntu4_all.deb
dpkg -i bcmwl-kernel-source_5.100.82.38+bdcom-0ubuntu4_amd64.deb 
All the packages were installed successfully. Now, the Broadcom STA driver was visible in 'Additional Drivers'. WiFi was working after enabling the driver (You might have to restart) .

If you have the "Wireless is disabled by hardware switch" message when you click the network icon, make sure that if you have a switch on the side of your laptop, it is turned on. Also, press the wireless key on the keyboard which is mostly F2 or 'Fn + F2' to turn On wireless. More details here.
