Open Source Workshop by LUG-VIT

Mar 13th, 2012

The Linux Users Group of VIT University is going to conduct a Open Source Workshop on March 17th.


Topics to be covered : 1. Introduction to Open Source and LUG 2. How to contribute to open source projects 3. Python Programming 4. Android Development 5. KDE 6. MediaWiki 7. Windows Phone 7 Development 8. Version Control - Git 9. How to earn through open source No Registration Fees. This event is open to all.
