Remove Duplicates From a List in Python

Mar 18th, 2012

The fastest method I came across which doesn’t disturb the order/sequence of the list is this:
def f7(seq):
    seen = set()
    seen_add = seen.add
    return [ x for x in seq if x not in seen and not seen_add(x)]
