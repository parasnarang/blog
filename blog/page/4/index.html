Cricket Fever.. Not So Much This Time!

Feb 22nd, 2011

I am a big Cricket fan, atleast I used to be one. It is said that in India, Cricket is a Religion. So, this time of the year i.e The ICC Cricket World Cup is more like a big festival out here! India being one of the hosts just adds to this.

[Image Source]
Unfortunately, I am missing out the on the fun for now. Exams are on full swing in my university. 7 days in a row. No break. Sad right? Anyway not the point. In my hostel, the T.V room is locked during exam time. ESPN Cricinfo is the best option to stay in touch with the live scores. And of course all of India’s matches are streamed live on the EspnStar website. Live streaming is a not-so-bad experience thanks to the decent wifi speed. I am eager to try out streaming with the recently launched Airtel 3G.
Come Sunday, It will be done :)
[Sunday = India v/s England]
Link : www.espnstar.com/cwclive
Also, there are many websites hosting competitions like the Fantasy Premier League for Cricket too. I came across one such site with a similar game for this World Cup. I really like its neat and simple interface. It is developed using Ruby on Rails and deployed on Heroku. It requires a facebook account to log in and play.
Link : www.rummy.openbala.com
I also came across a ‘cricket score readout script’ in a lug mailing list. The script speaks out loud the latest live score. It reads the scores from cricinfo in the background. You need eSpeak installed for it to work.
Link : www.sourceforge.net/projects/sayscore/files/sayscore_wc11.sh/download

Well, I think I should get back to concentrating on my exams. Desperately waiting for these 7 days to get over ASAP and join in the cricket craze! Who wants to miss out on a blazing Sehwag?

My First Facebook App

Jan 31st, 2011

Now-days I am learning to work with Php & MySQL on the xampp server.
Learning without actual implementation doesn’t make sense. So I started making a list of things that I can do using Php.
I came across the facebook-php sdk. It seemed like fun. Plus facebook with its APIs provides a nice platform to work on.
So finally, after 2 nights of work, I finished my first facebook app. Using my app, students from my university can check their semester attendance from facebook itself.


App name: VIT University - Check your attendance
Link: http://apps.facebook.com/vitattendance/

Here is the official update about this app from my college.

Developing facebook apps is very easy. A good amount of quality documentation and sample code are available in the forums. Spend some time on the documentation and in understanding the basics of their php-sdk, javascript-sdk, fbml and the awesome graph API and then all you will need is a good idea.

Worst Experience Installing Linux Till Date

Dec 27th, 2010

Currently I have a dual boot of Windows 7 and Ubuntu 9.10. 
I hardly use the ubuntu partition. Instead i use Fedora 13 on a VirtualBox on Windows 7.
I have been planning to update the other partition for a long time now, but thanks to VOLSBB, I didn’t yet. It’s a monday morning and I have nothing better to do. Lets update :)

[Hardware Config:  
Processor: Intel Core2Duo P8700 @ 2.53 GHz
RAM: 4GB DDR2
Graphic Card: ATI Mobility Radeon HD 4570.]

7:00 AM :
Thinking of installing Fedora 14 on the Ubuntu partition. I have the Fedora 14 x86_64 DVD iso image. 3.27 GB. Time to put the 8 GB Transcend USB drive (which i got for free at the Microsoft stall in Chennai @ OSIDays this year) to some good use.
Making live usb using Unetbootin.

7:10 AM :
Live USB created. Now time for installation. Rebooting. F-12. Selected Boot from USB.

7:15 AM :
At the last step of installation the following error pops up:-
———————————————————————————–
Missing ISO 9660 Image


The installer has tried to mount image #1, but cannot find it 
on the hard drive.


Please copy this image to the drive and click Retry. Click
Exit to abort the installation.
———————————————————————————–

What??!! Nothing happens when i retry & clicking exit rebooted the system.
After restarting:-
———————————————————————————–
GRUB Loading…
error : file not found
grub rescue >
———————————————————————————–

I guess GRUB was in the ubuntu partition which got formatted during installation. Dunno what to do now. Can’t even log into Windows. So no google help either :(
Looking for help. Found the Ubuntu 9.10 live cd. Booting from it. How to install just GRUB? Not able to install wifi driver while running live, so no Google yet. Can’t think of any option other than installing Ubuntu again.

7:50 AM :
Installation taking time. Reading newspaper in the meantime. Australia allout for just 98!! wow!

8:00 AM :
Finally installation completed. Rebooting. Grub menu loaded. Yayy!
Logging in to windows7. Finally Google! Searching the fedora forums to find a solution for the error. Looks like I have to copy the iso image to the root folder of the USB after the creation of live USB. Also, it has been suggested to use Linux Live USB creator or Fedora liveusb-creator.

8:10 AM :
Downloading Linux Live USB creator 2.6. Download speed of 190 kb/s, pretty good for wifi here. :)
Nice colorful interface! Creating live USB.

8:20 AM :
Live USB creation failed. WHY?? Maybe some problem with the iso image. Checking sha256sum. Matches. Then why the error? Should try Fedora liveusb-creator instead.

8:30 AM :
Creation success. Rebooting. F-12. Boot from USB. The screen gets stuck at:-
SYSLINUX 3.85 something something Copyright (c) 1994-2010 H. Peter Anvin et al

What now? 
Rebooting. Logged in to windows7. Google. This is another common error on fedora forums. Will have to update the Syslinux of liveusb-creator to 4.02 or above. Found the solution here. Have to replace the syslinux.exe file with the new one at C:\Program Files\LiveUSB Creator\tools and then run liveusb-creator again. Simple. I love the fedora community!
Done with the updates. Created live USB again. Reboot. F-12. Boot from USB. Anaconda loads. :)

9:15 AM :
Almost at the end of installation and again the Missing ISO 9660 image error. I forgot to copy the iso image to root folder of my usb drive. Damn!
Reboot. GRUB missing. AAAhhhrrghh!!!
Booting from Ubuntu 9.10 live cd. Copying iso image to root folder of USB.
Done. Reboot. F-12. Boot from USB. Installation process.

10:05 AM :
No missing ISO 9660 error this time! :)
Selecting Packages to install. Done. Checking repos. Gives yet another error. NO!! Not now.. grr. Error:-
yum-utils.something.noarch not found. Check the Installation Source.
Reboot or Retry.
Retry displays same thing again.
Rebooting. Installation progress vanishes. Grub not found. Just sad.
10:35 AM :
Have to go out with friends after an hour. Should get the computer working for now. Booting from Ubuntu 9.10 live cd. Not in a mood to install 9.10 again.
Just realized I also have the iso image of Maverick Meerkat.
Using default “Startup Disk Creator” to create live USB for Ubuntu 10.10.
Reboot. F-12. Boot from USB. Hope this works.

11:15 AM :
Success! Logging in to Ubuntu 10.10 Maverick Meerkat. Wifi settings configured. Net working! Now dual boot of Windows 7 and Ubuntu 10.10.

Made lots of mistakes today. Realized that incomplete knowledge does more harm than good. Almost 4 hours spent. Proper knowledge & understanding as well as patience is required. Lesson learned the hard way.
Have to go out now. Will try installing Fedora again tonight. But this time after some research and under guidance!

Browser Wars: Is Your Browser Prepared for HTML5 ?

Dec 22nd, 2010

HTML5 is going to be the next big thing! It is currently under development and will be the next major revision of the HTML standard. W3C and WHATWG are currently working on its development. 


Basically it is going to make life very easy, both for developers as well as end users. Once HTML5 is approved, there wont be any need of proprietary players or of Flash, Silverlight etc. All the media will be a normal part of the webpage. Notable examples of HTML5 are TweetDeck app for Google Chrome, YouTube’s HTML5 videos site. Adobe html5 pack for dreamweaver has been released. Apple.com also has a HTML5 page.


I conducted the Html5 test using www.html5test.com on the current popular web browsers and the results are as follows:
